# Parallel Editing Agent Demo

A collaborative document editing environment where human users and AI agents work together in real-time.

## Features
- **Real-time Collaboration**: Powered by Yjs and WebRTC.
- **AI Agents**: Orchestrate complex edits using LLMs (Live or Mock modes).
- **Parallel Execution**: Multiple agents can edit different sections simultaneously.
- **Conflict Resolution**: Smart merge and conflict detection.

![Demo Screenshot](screenshot.webp)

## Setup
1. Open `index.html` in a modern browser (or serve via a local server).
2. Configure LLM settings in the "Settings" modal for Live AI.
3. Use Mock AI mode for testing without API keys.
